早晚安图片生成-pikachu

接口详情: out_image.php:POST/GET
Github地址   批量导入图片   批量导入文本

字段 字段类型 字段介绍 取值范围 字段可空
img string 背景图片 URL 可空
w float 生成图片的宽 [400,1080] 可空
h float 生成图片的高 [500,1920] 可空
t1 string 问候语 （如：早上好鸭，晚安）（其他字需要ttf字体支持） (0,4]个中文 可空
t2 string 英文问候语 （如：GOOD NIGHT）（早安图片不生效） (0,10]个英文 可空
t3 string 励志文字 （多余文字会自动丢弃） (0,50]个中文 可空
g1 float 外边框模糊度 （晚安图片不生效） [0,6] 可空
g2 float 背景图片糊度 [0,6] 可空
mr float 卡片大小 （晚安图片不生效） [0,0.4] 可空
na int 上下白边透明度 （早安图片不生效） [0,127] 可空
cc int 二维码内容 (0,6000]字符 可空
cl string 二维码中间log图片 URL 可空
ct string 二维码下方文字 [0,6]个中文 可空
ca int 二维码透明度 [0,100] 可空
