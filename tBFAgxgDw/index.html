百度网盘在线直链解析不限速下载：搭配 IDM + ARIA2
Windows · 2021年4月22日
从阿里云盘公测后，D盘可能为了避免流失一些用户，就偷偷发布了安卓内部精简版，速度方面比较良心，并且没有乱七八糟的功能。另外之前锋哥也给大家分享过 PC 端的旧版D盘客户端，速度上也还行。

不过使用这两个版本，前提是你的D盘账号不是黑号，也就是之前没有在第三方客户端上登陆过，否则的话，除了充值会员，任何客户端版本下载速度都是龟速。

那么黑号就无解了？今天给大家分享在线直链高速下载的方法，可以无需登陆账号，无需下载客户端，通过分享链接就可以获取高速下载地址。（注：下面提到的软件和网站，都会在文末提供哦！）

某盘高速下载

方法1.安装 User-Agent 扩展插件

因为D盘的限制，首先你的浏览器 Chrome / EDGE 或者其它基于 Chromium 内核的浏览器需要安装 User-Agent 扩展来修改浏览器的标识。

安装好后，右键点击 User-Agent 扩展 → 选项，填入下面的内容。
'''
New User-Agent name：D盘分享下载
New User-Agent String：LogStatistic
Group：D盘分享
Append?：Replace
Indicator Flag：Log
点击 Add 保存
'''

百度网盘在线直链解析不限速下载：搭配 IDM + Aria2-1

接着打开在线解析网站，输入文件分享的地址，点击获取下载地址。注意，每个 IP 每天只有 20 次的获取。然后点击下载链接，就可以直接下载了。

方法2：使用 Aria2 下载

下载好 Aria2 工具后，运行 "点此启动.bat"，然后如上面的分享文件解析操作，点击 "推送到Aria2"。

可以看到 Aria2 下载速度比使用 User-Agent 扩展浏览器下载的要快多了，这里锋哥测试，接近达到带宽的极限了，稳定在 28 MB/s 左右。

方法3：使用 IDM 下载工具

IDM就不用多介绍了吧，著名的多线程下载工具，下载好 IDM 后，需要先设置下 UA 参数。

打开 IDM 菜单 → 下载 → 选项，找到下载项目，手动添加任务时使用的用户代理（UA）→ 填入 LogStatistic，然后点确定。如下图。

百度网盘在线直链解析不限速下载：搭配 IDM + Aria2-5

然后还是到那个解析下载界面里面，右键复制 "下载链接" 的链接地址，到 IDM 里面新建立下载，粘贴地址。

你没看错，速度在 40 MB/s，锋哥是 300M 的带宽，这比官方 SVIP 的下载速度还要快，过瘾！

总结
对于临时急需下载D盘分享资源来说，这个网站的确可以帮上你的大忙，但是这个网站可能为了避免流量过大，每天对访问的 IP 有次数限制（在网站界面会有红绿灯提示速度状态），不过这应该也不是啥大问题，换个 IP 就搞定。
